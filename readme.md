# SPA-dashboard

![spa-dashboard](/public/img/spa-dashboard.png)

## stack technic
  - Node
  - Express
  - MongoDB

## front-end
  - Bootstrap
  - List.js

## Instalation

- Clone repository on your desktop
- if not installed,[add MongoDB](https://docs.mongodb.com/manual/installation/)
- Start mongo
- open to terminal :

```
cd /pathToFolder/spadashboard
npm install
node index.js
```
